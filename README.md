# 🇮🇹 Guida turistica valencia - progetto scolastico

Questa è una Web App interattiva creata per accompagnare gli studenti durante la gita scolastica a Valencia (Aprile 2026). 
L'app include una guida storica, i simboli della città, una mappa interattiva con i punti di interesse e il programma di massima del viaggio.

## CARATTERISTICHE
- **Multilingue:** supporto completo per Italiano e Spagnolo.
- **Mappa interattiva:** gestita con Leaflet.js per individuare monumenti, hotel e ristoranti.
- **Programma interattivo:** diario di viaggio con link diretti alle sezioni della guida.
- **_Mobile first_:** ottimizzata per l'uso su _smartphone_ durante le camminate in città.

## CONTRIBUIRE
Il progetto è rilasciato sotto **licenza MIT**, il che significa che è aperto a tutti! 

## AIUTARE
Se sei un docente o uno studente e vuoi usare questa guida per il tuo viaggio:
1. **Fai un Fork:** crea una tua copia del progetto per personalizzare l'itinerario o le foto.
2. **Segnalazioni:** se trovi un errore nelle traduzioni o un link rotto, apri una "Issue".
3. **Migliora il codice:** se aggiungi nuove funzioni (es. quiz, nuove sezioni, integrazioni video), invia una "Pull Request". Sarò felice di integrare i tuoi miglioramenti nella versione principale!

## LICENZA
Distribuito sotto licenza MIT. Vedi il file `LICENSE` per maggiori dettagli.

# 🇬🇧 Valencia tourist guide - school project

This is an interactive Web App created to accompany students during their school trip to Valencia (April 2026). 
The app includes a historical guide, city symbols, an interactive map with points of interest, and the general travel program.

## FEATURES
- **Multilingual:** full support for Italian and Spanish.
- **Interactive map:** managed with Leaflet.js to identify monuments, hotels, and restaurants.
- **Interactive program:** travel diary with direct links to the guide sections.
- **_Mobile first_:** optimized for _smartphone_ use during city walks.

## CONTRIBUTING
The project is released under the **MIT license**, which means it is open to everyone!
